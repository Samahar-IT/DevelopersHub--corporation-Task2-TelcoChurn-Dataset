Objective

To build a machine learning pipeline that predicts whether a customer will churn using structured telecom data, helping businesses improve customer retention strategies.

Methodology / Approach
Dataset: Telco Customer Churn dataset
Data preprocessing using ColumnTransformer
Handling missing values
Encoding categorical features
Scaling numerical features
Models used: Logistic Regression, Random Forest
Hyperparameter tuning using GridSearchCV
End-to-end ML pipeline built with Scikit-learn for automation and reusability
Key Results / Observations
Random Forest outperformed Logistic Regression after tuning
Pipeline improved workflow efficiency and model consistency
Feature engineering significantly impacted performance
Model achieved improved accuracy with optimized parameters
